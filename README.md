# ASAB

In this git the preliminary results along with the code to reproduce them can be found.

## CVAutoEncoder
This folder contains all the Code needed to train the AutoEncoder.

## DATA
This folder contains the data for the next experiment planned, the runway detection model.

## MonteCarlo
This folder contains basic Monte Carlo sampling scripts utilized to generate the adversarial input samples to trick the software under test.

## SUT
This folder contains everything needed for the software under test. Training and testing scripts. Preliminary results can be found here as well.
